# 凯哥 逆水寒-科举答题助手
- 截图识别（OCR）题目，效率更高
- 不怕拼诗句

## Demo
![效果图](./demo.jpg)

## 技术方案
- 客户端 C#(WinForm)
- 后端 Java(springboot)
- DB postgresql
- OCR用的百度的API
- 普通题是查题库，拼诗词是从85W首古代诗词全集中，正则扫库的
